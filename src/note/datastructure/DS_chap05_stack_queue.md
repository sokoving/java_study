# 스택과 큐는 리스트와 달리 중간 삭제, 삽입이 없다

# 스택(stack)
- 자료를 차곡차곡 쌓아놓은 형태
  + link는 선형 구조 
- LIFO(Last-In-First-Out) 구조
  + 후입 선출 방식(나중에 넣은 걸 먼저 뺀다)
1. 삽입(push, insertion)
2. 제거(pop, deletion)
3. 읽기(peek)

# 큐(queue)
- 자료를 줄지어 연결해놓은 형태

## 우선순위 큐
- 자료가 정렬되어 들어간다
